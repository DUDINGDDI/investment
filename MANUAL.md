## 1. 행사 전체 흐름

### 타임라인

```
[입장] 사번 + 이름으로 로그인
  ↓
[오전] 하고잡이 투자 (AM)
  - 33개 부스를 돌며 QR 방문 → 평가 → 주식 매수/매도
  - 아이디어 제안, 리뷰 작성
  - 미션 수행
  ↓
[오후] 대표작 투자 (PM)
  - 11개 대표 부스에 코인 투자/철회
  - 메모 작성
  ↓
[마감] 결과 공개
  - 부스 순위, 시상 결과 발표
  - 티켓 교환
```

### 화면 구성

- **AM 탭** (상단): 미션홈 | 부스목록 | 거래내역
- **PM 탭** (상단): 투자홈 | 부스목록 | 투자내역 | 지도 | 마이
- **플로팅 메뉴**: AM↔PM 전환, QR 스캔, 결과 보기
- **공지 배너**: 관리자가 등록한 공지가 실시간으로 상단에 표시

## 2. 참가자 유형별 차이

### 신입사원 (Rookie)

행사의 주요 참가자입니다.

| 항목 | 내용 |
|------|------|
| **PM 초기 잔액** | 1억원 |
| **AM 초기 잔액** | 1억원 |
| **PM 투자 제한** | 자기 계열사 부스에는 투자할 수 없음 (예: CJ올리브영 소속이면 CJ올리브영 부스 투자 불가) |
| **AM 투자 제한** | 자기 소속 부스에 매수/매도 불가, 부스당 최대 3천만원 |
| **미션** | 6개 미션을 직접 수행해야 함 |
| **티켓** | 미션 완료 시 키캡 교환권 획득 |
| **시상 대상** | 대부분의 상이 신입사원 대상 |

**자기 계열사 제한이 걸리는 원리:**
- PM: 로그인한 사용자의 `소속 회사`와 부스의 `카테고리(계열사)`가 같으면 투자 버튼이 비활성화됩니다.
- AM: 사용자에게 `소속 AM 부스`가 지정되어 있으며, 해당 부스는 투자 탭 자체가 잠깁니다.

### 경영진 (임원)

부스를 평가하고 투자하는 참여자입니다.

| 항목 | 내용 |
|------|------|
| **PM 초기 잔액** | 10억원 (신입의 10배) |
| **AM 초기 잔액** | 1억원 |
| **PM 투자 제한** | 없음 (자기 계열사 포함 자유 투자) |
| **AM 투자 제한** | 부스당 금액 제한 없음 (자기 소속 부스만 불가) |
| **미션** | 모든 미션 자동 완료 (바이패스) |
| **티켓** | 자동 완료이므로 바로 교환 가능 |
| **시상 대상** | "Executive 순위"에만 반영 |

**임원의 투자가 중요한 이유:**
- 대표작 결과에서 `Executive 순위`는 임원 투자금만으로 산정됩니다.
- `전체 순위`는 Rookie 점수 + 임원 투자금의 합산이므로, 임원 투자가 최종 순위에 직접 영향을 줍니다.

### 팀장 & 인재원

행사 운영을 지원하는 참여자입니다.

| 항목 | 내용 |
|------|------|
| **PM 초기 잔액** | 1억원 |
| **AM 초기 잔액** | 1억원 |
| **PM 투자 제한** | 없음 (자기 계열사 포함 자유 투자) |
| **AM 투자 제한** | 부스당 최대 3천만원, 자기 소속 부스 불가 |
| **미션** | 모든 미션 자동 완료 (바이패스) |
| **랭킹 반영** | Rookie도 Executive도 아니므로 대표작 결과 순위에 반영되지 않음 |

**팀장/인재원의 투자는 어떻게 처리되나?**
- PM 부스의 "총 투자금"에는 합산되지만, 대표작 결과의 Rookie/Executive 점수에는 포함되지 않습니다.
- AM 부스의 "총 보유량"에는 합산됩니다.
- 즉, 부스 간 단순 랭킹에는 영향을 주지만, 최종 시상용 점수에는 영향이 없습니다.

## 3. 오전 프로그램: 하고잡이 투자 (AM)

33개 AM 부스를 돌며 주식처럼 매수/매도하는 프로그램입니다.

### 참가자가 하는 일

```
1. 부스 방문 → QR 스캔 (방문 기록)
2. 부스 평가 → 6항목 별점 (1~5점)
3. 주식 매수/매도 → 1만원 단위
4. (선택) 리뷰 작성, 아이디어 제안
```

### 거래 조건

매수/매도를 하려면 **반드시 아래 두 가지를 먼저 완료**해야 합니다:

1. **부스 방문** (QR 스캔)
2. **부스 평가** (6항목 별점 제출)

방문만 하고 평가를 안 하면 투자 탭이 잠금 상태입니다.

### 거래 규칙

- 거래 단위: **1만원** (1만원 미만 거래 불가)
- 초기 잔액: **1억원** (모든 참가자 동일)
- 신입사원: 부스당 **최대 3천만원** 투자
- 임원: 부스당 **금액 제한 없음**
- **자기 소속 부스**: 매수/매도 불가 (투자 탭 자체가 잠김)

### 주식 가격

- 각 부스마다 주식 가격이 있으며, 관리자가 행사 중에 변경할 수 있습니다.
- 가격 변경 이력이 차트로 표시됩니다.

### COSPI 지수

전체 시장의 활성도를 보여주는 지표입니다.
```
COSPI = 전체 참가자의 누적 매수 총액 - 누적 매도 총액
```
AM 홈 화면에서 실시간으로 확인할 수 있습니다.

### 메모 기능

각 부스에 대한 개인 메모를 작성할 수 있습니다. AM 부스 메모는 **기기(브라우저)에 저장**되므로, 다른 기기에서는 보이지 않습니다.

## 4. 오후 프로그램: 대표작 투자 (PM)

11개 PM 부스에 가상 코인을 투자하는 프로그램입니다.

### 참가자가 하는 일

```
1. 부스 목록 확인 (이름, 계열사, 설명)
2. 투자하기 → 금액 입력 → 잔액에서 차감
3. 철회하기 → 투자한 금액 중 일부/전액 회수
4. (선택) 부스별 메모 작성
```

### 투자 규칙

- 투자 단위: **1만원**
- 초기 잔액: 신입 **1억원**, 임원 **10억원**
- 신입사원: **자기 계열사 부스 투자 불가**
- 임원/팀장: 모든 부스에 자유 투자
- 철회: 투자한 금액 이하로만 가능

### "신입사원 Pick!" 부스

PM 11번째 부스는 특별한 부스입니다.
- 관리자가 오전 AM 부스 중 하나를 선택하면, 해당 부스의 정보(이름, 계열사, 설명)로 교체됩니다.
- 이미 투자된 금액은 그대로 유지됩니다.
- 관리자 페이지 → 기능 관리 → "신입사원 Pick 부스 변경"에서 설정합니다.

### 메모 기능

PM 부스 메모는 **서버에 저장**됩니다. 어떤 기기에서든 확인 가능하며, 마이페이지에서 모아볼 수 있습니다.

## 5. 미션 시스템

신입사원이 행사 중 수행하는 6개 미션입니다. 각 미션을 완료하면 **키캡 교환권 티켓**을 받습니다.

### 미션 목록

| 미션 | 이름 | 어떻게 완료하나 |
|------|------|----------------|
| **내일 더 새롭게** | renew | 현장에 설치된 QR 코드를 스캔 |
| **꿈을 원대하게** | dream | AM 부스에 70자 이상 아이디어 제안을 **5개** 작성 |
| **반드시 결과로** | result | 자동 완료 (전원 사전 지급) |
| **안돼도 다시** | again | 내 AM 부스 방문자 수가 **70명** 이상 |
| **진정성 있게** | sincere | 리뷰를 포함한 부스 평가를 **12개** 이상 제출 |
| **함께하는 하고잡이** | together | 현장에 설치된 QR 코드를 스캔 |

### 미션별 상세 설명

**"내일 더 새롭게" (renew)**
- 현장 벽에 부착된 QR 코드를 스캔하면 완료됩니다.
- 완료 시 **AM 주식 계좌에 1억원이 추가 지급**됩니다. (오전 투자 자금 추가)

**"꿈을 원대하게" (dream)**
- AM 부스 상세 → "꿈을 원대하게" 탭에서 아이디어를 작성합니다.
- **70자 이상** 작성해야 1건으로 인정됩니다.
- 5건 달성 시 완료. 진행도가 실시간으로 표시됩니다.
- 70자 미만으로 수정하면 해당 건이 카운트에서 빠집니다.

**"안돼도 다시" (again)**
- 다른 참가자들이 내 AM 부스를 방문(QR 스캔)하면 카운트가 올라갑니다.
- 70명이 방문하면 완료.
- 직접 할 수 있는 미션이 아니라 **다른 참가자의 방문에 의존**합니다.
- PM 부스에 소속된 신입사원은 AM 부스가 없으므로, 로그인 시 자동 완료 처리됩니다.

**"진정성 있게" (sincere)**
- AM 부스 상세 → "진정성 있게" 탭에서 리뷰를 작성합니다.
- 평가(별점)를 제출한 후에만 리뷰 작성이 가능합니다.
- 리뷰는 **최소 20자** 이상이어야 합니다.
- 12개 부스에 리뷰까지 작성하면 완료.
- 리뷰를 삭제하면 카운트가 줄어듭니다.

**"함께하는 하고잡이" (together)**
- 현장 사진벽에 부착된 QR 코드를 스캔하면 완료됩니다.

### 미션 진행도 표시

AM 홈 화면에서 6개 미션 배지를 확인할 수 있습니다:
- 정량 미션(dream, again, sincere): 프로그레스 바로 진행률 표시
- 기타 미션: 완료/미완료 상태 표시
- 완료 시 배지가 활성화되고 축하 팝업이 뜹니다.

### 비신입사원 (임원, 팀장)의 미션

임원과 팀장/인재원은 모든 미션이 **자동으로 완료** 처리됩니다. 별도로 미션을 수행할 필요가 없습니다.

### 미션 랭킹

정량 미션(dream, again, sincere)에 대해 참가자 간 랭킹을 볼 수 있습니다.
- 관리자가 랭킹 공개/비공개를 설정합니다.
- 내 순위, 등락, 상위 10명이 표시됩니다.

## 6. 부스 평가 & 리뷰

AM 부스를 방문한 후 평가를 제출합니다.

### 평가 절차

```
1. 부스 방문 (QR 스캔)
2. 부스 상세 → [평가] 탭
3. 6개 항목에 별점 (1~5점) 부여
4. (선택) 리뷰 작성
5. 제출
```

### 평가 6개 항목

| 항목 | 질문 |
|------|------|
| 첫인상 | 부스의 첫인상이 어떠했나요? |
| 최고점 | 부스의 가장 좋은 점은? |
| 차별성 | 다른 부스와 차별화되는 점은? |
| 1위 전략 | 1위가 될 수 있는 전략이 있나요? |
| 격차 극복 | 부족한 점을 극복할 수 있나요? |
| 글로벌 | 글로벌 가능성이 있나요? |

### 평가와 투자의 관계

- **평가를 해야 투자**할 수 있습니다. (방문 + 평가 = 투자 잠금 해제)
- 평가 없이 매수/매도를 시도하면 "평가를 먼저 완료해주세요" 메시지가 표시됩니다.

### 리뷰

- 평가(별점) 제출 후에만 리뷰를 작성할 수 있습니다.
- 최소 **20자** 이상.
- 수정/삭제 가능합니다.
- 리뷰를 작성하면 "진정성 있게" 미션 진행도에 반영됩니다.

## 7. 아이디어 보드

AM 부스에 개선 아이디어나 제안을 작성하는 공간입니다.

### 작성 규칙

- 부스 방문(QR 스캔) 후에만 작성 가능
- **최소 70자** 이상 (미션 연동)
- 본인 글만 수정/삭제 가능
- 수정 시 70자 미만이 되면 "꿈을 원대하게" 미션 카운트에서 제외

### 미션 연동

70자 이상 아이디어를 **5개** 작성하면 "꿈을 원대하게" 미션 완료.

### 실시간 갱신

다른 참가자가 아이디어를 작성하면 실시간으로 목록이 갱신됩니다 (아이디어 보드 페이지).

## 8. 티켓 교환 시스템

미션을 완료하면 **키캡 교환권**을 받습니다.

### 교환 가능 미션 (5개)

| 미션 | 완료 시 교환권 |
|------|---------------|
| 내일 더 새롭게 | 키캡 1개 |
| 꿈을 원대하게 | 키캡 1개 |
| 안돼도 다시 | 키캡 1개 |
| 진정성 있게 | 키캡 1개 |
| 함께하는 하고잡이 | 키캡 1개 |

- "반드시 결과로"는 교환권이 없습니다.
- 사진 티켓(`photo_*`)은 별도 교환 대상이 아닙니다.

### AI 사진 세션

미션 3개 이상 완료하면 **AI 사진 세션 참여 QR**이 추가로 표시됩니다.

### 교환 절차

```
1. 참가자: 마이페이지 → [티켓] 탭 → 사용할 티켓 클릭 → QR 코드 표시
2. 관리자: 티켓 스캔 페이지에서 QR 스캔 → 사용 처리
3. 참가자: 키캡 수령
```

- 개별 티켓 사용 또는 "전부 사용" 일괄 처리 가능
- 사용된 티켓은 화면에서 "사용됨" 상태로 변경

## 9. 랭킹 & 시상

### PM 부스 랭킹 (대표작 투자)

모든 참가자의 투자금 합계로 부스 순위를 매깁니다.
- 동점 시: 먼저 해당 금액에 도달한 부스가 상위

### AM 부스 랭킹 (하고잡이 투자)

모든 참가자의 주식 보유량 합계로 부스 순위를 매깁니다.

### 대표작 결과 (최종 순위)

PM 부스의 최종 순위는 **신입사원 점수 + 임원 투자금**으로 결정됩니다.

**신입사원(Rookie) 점수 공식:**
```
Rookie 점수 = (해당 부스의 신입 투자금 합계) ÷ (304 - 해당 계열사 신입 인원수) × 234
```
- 이 공식은 계열사별 인원수 차이를 보정합니다.
- 인원이 많은 계열사가 유리하지 않도록 나눗셈으로 정규화합니다.

**3가지 순위:**

| 순위 | 기준 | 용도 |
|------|------|------|
| **전체 순위** | Rookie 점수 + 임원 투자금 | 최종 대표작 순위 |
| **Rookie 순위** | Rookie 점수만 | 신입사원 투표 결과 |
| **Executive 순위** | 임원 투자금만 | 임원 투표 결과 |

### 시상 (9개 카테고리)

| 상 | 누가 받나 | 기준 |
|----|----------|------|
| **11번째 발표 부스** | AM 부스 1곳 | AM 부스 총 투자금 1위 |
| **하고잡이 투자자상** | 신입 1명 | 각 AM 부스에서의 투자 순위를 가중치로 환산한 총점 |
| **문어발 투자자상** | 신입 1명 | AM에서 투자한 부스 가짓수가 가장 많은 사람 |
| **발품투자자상** | 신입 1명 | AM 부스를 가장 많이 방문한 사람 |
| **얼리버드 투자자상** | 신입 1명 | 6개 미션을 가장 먼저 모두 완수한 사람 |
| **환승투자자상** | 신입 1명 | PM 투자에서 철회 횟수가 가장 많은 사람 |
| **막차투자자상** | 신입 1명 | PM/AM 통틀어 마지막 투자 시각이 가장 늦은 사람 |
| **꿈을 원대하게상** | 신입 1명 | 아이디어 댓글을 가장 많이 쓴 사람 |
| **안돼도 다시 상** | AM 부스 1곳 | 방문자가 가장 많은 AM 부스 |

**하고잡이 투자자상 가중치:**
```
AM 부스별 투자 순위에 따라 점수 부여:
1위=10점, 2위=9점, 3위=8점, 4위=7점, 5위=6점,
6~7위=5점, 8~10위=4점, 11~15위=3점, 16~22위=2점, 23위~=1점

→ 모든 부스의 점수를 합산한 총점이 가장 높은 사람
```

## 10. 관리자 운영 가이드

### 접속 방법

- URL: `https://도메인/admin`
- 비밀번호: `rolypoly` (전체 기능) / `onlyonefair` (미션 QR 코드만)

### 행사 전 준비

1. AM 거래 활성화 확인 (기능 관리 → "하고잡이 투자" 활성)
2. PM 투자 비활성화 (오전에는 PM 투자를 막아둠)
3. 결과 비공개 상태 확인
4. 공지사항 초기 메시지 등록

### 오전 운영 (AM 시간)

| 작업 | 위치 | 설명 |
|------|------|------|
| 공지 등록/변경 | 상시기능 → 공지사항 | 실시간으로 전체 참가자에게 전달됨 |
| 주식 가격 변경 | (별도 API) | 행사 분위기에 따라 부스별 주식 가격 조정 |
| 미션 QR 부착 | 상시기능 → 미션 QR | QR 이미지를 현장에 부착 |
| 투자 순위 확인 | 투자 순위 탭 | AM 부스 실시간 순위 확인 |

### 오전→오후 전환

1. **AM 거래 중단**: 기능 관리 → "하고잡이 투자 중단"
2. **신입사원 Pick 부스 설정**: 기능 관리 → AM 부스 중 하나 선택 → 적용
3. **PM 투자 활성화**: 기능 관리 → "대표작 투자 활성화"
4. **공지 변경**: "오후 대표작 투자가 시작되었습니다!" 등

### 오후 운영 (PM 시간)

| 작업 | 위치 | 설명 |
|------|------|------|
| PM 투자 순위 확인 | 투자 순위 탭 | PM 부스 실시간 순위 |
| 대표작 결과 확인 | 대표작 결과 탭 | Rookie/Executive/전체 순위 |
| 보고서 생성 | 상시기능 → 보고서 | 현 시점 스냅샷 저장 (경영진/신입/종합) |

### 마감 & 결과 발표

1. **PM 투자 중단**: 기능 관리 → "대표작 투자 중단"
2. **보고서 생성**: 상시기능 → 보고서 생성 (최종 결과 저장)
3. **결과 공개**: 기능 관리 → "결과 공개하기"
4. **시상 확인**: 시상 탭에서 9개 카테고리 수상자 확인

### 티켓 교환 운영

- **상시기능 → 티켓 스캔** 클릭 → 카메라로 참가자 QR 스캔
- 개별 티켓 또는 일괄 사용 QR 모두 인식 가능
- 스캔 결과 화면에서 사용 처리 완료 확인

### 비상 조치

| 상황 | 대응 |
|------|------|
| 참가자 투자가 안 됨 | 기능 관리에서 해당 투자가 "활성" 상태인지 확인 |
| 미션이 완료 안 됨 | 관리자가 특정 미션을 전체 일괄 완료 처리 가능 |
| 잘못 일괄 완료한 미션 | 관리자가 해당 미션 일괄 취소 가능 |
| 공지가 안 내려감 | 공지 삭제 버튼 클릭 |
